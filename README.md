# 🍉 pkuhollow 1st Anniversary | 未名树洞一周年生贺

> *"这一年，越过有形的围墙，我们在未名的荒野里，种下了一颗会发光的西瓜。"*

这是一个专门为 **新未名树洞一周年** 制作的沉浸式纪念网页。祝新未名树洞生日快乐！这份生日礼物，也献给每一位洞友。

## ✨ 设计与交互

- **星空背景**：使用 HTML5 Canvas 渲染的星空，象征自由且无界的旷野。
- **发光西瓜**：SVG 手绘的树洞标识，配合 CSS3 动画实现悬浮效果。
- **星星盲盒**：点击背景中的星星，不仅会荡开涟漪，还会随机遇见一位“洞主”或“洞友”（可能解锁神秘人物哦！😋）。
- **坐标信号**：左下角未名湖经纬度（39°59'23.0"N 116°18'19.0"E），点击坐标可触发信号连接。
- **随机流星**：在静谧的浏览时刻，会有流星随机划过夜空，增加沉浸感。
- **标签页联动**：当用户切换标签页离开或归来时，网页标题都会随之变化。

## 🛠️ 技术实现

- **Vanilla JavaScript**：负责 Canvas 动画逻辑、星空渲染及交互响应。
- **CSS3 Animations**：处理所有平滑过渡、呼吸灯及流星特效。
- **Tailwind CSS**：用于构建响应式布局。
- **单文件架构**：无外部图片依赖，轻量化部署，支持所有现代浏览器。
